## Getting started

<Steps>

1. **Sign up**

   Go to [synatrahq.com/login](https://synatrahq.com/login) and create an account.

2. **Create an agent**

   Navigate to Agents and create your first agent. Configure the model, system prompt, and assign tools.

3. **Start a conversation**

   Go to the Inbox, select a channel, and start a new thread with your agent.

</Steps>
